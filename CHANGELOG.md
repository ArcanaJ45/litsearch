# LitSearch 更新日志

## v1.1.0 (2026-03-02)

### ✨ 新功能
- **期刊影响因子 (IF)**: 通过 OpenAlex API 自动获取期刊 2 年平均被引次数（等效 JCR Impact Factor），结果展示影响因子数值与等级标识
- **课题相关性分析**: 用户可输入自己的课题描述，系统自动计算每篇文献与课题的相关度分数 (0-100)，基于 TF-IDF 余弦相似度与关键词覆盖率算法
- **智能内容摘选**: 从摘要中自动提取与用户课题最相关的句子，生成参考建议，指出文献可为用户研究提供哪些方面的帮助（研究方法、统计分析、生物指标、健康效应等）
- **文献详情增强**: 详情窗口新增影响因子、相关度评分、智能分析报告
- **版本信息与联系方式**: 界面底部显示版本号、开发者邮箱联系方式
- **开发者署名**: 顶栏与底栏显示开发者标识 "ArcanaJ"，联系信息字体放大至 10pt

### 🔧 改进
- 数据模型新增 `impact_factor` 和 `relevance_score` 字段
- CSV/TXT 导出支持影响因子和相关度信息
- 结果表格新增影响因子列和相关度列，支持排序
- 统计栏增加影响因子均值和相关度均值显示
- 检索输入框和课题描述框改为弹出式对话框输入，点击预览区域弹出大尺寸编辑窗口，支持滚动条、字数统计、Ctrl+Enter 快捷确认
- **UI 全面自适应**：左右面板改用 PanedWindow 可拖拽分割，用户可自由调整左右比例
- 左侧控制面板内容宽度随面板拖拽自动适配，预览标签文字换行宽度动态调整
- 表格标题/作者/期刊/DOI 列设为可拉伸 (stretch)，随窗口变大自动填充空间
- 最小窗口尺寸降至 800×550，支持更小屏幕使用
- 左侧控制面板改为可滚动布局，窗口较小时也能访问所有选项
- 移除检索输入区的"示例"快捷标签，精简界面
- 数据源选项改为竖排排列，每个选项独占一行，可正常点击"仅 PubMed"和"仅 Crossref"
- 结果数量新增 Spinbox 自定义输入（1-100），快捷按钮保留 10/20/30/50/100
- **弹出对话框布局修复**：按钮区域改为先于文本区 pack（side=BOTTOM），确保"✅ 确认保存"、"🗑 清空"、"取消"按钮始终可见；按钮更名为"确认保存"，增大点击区域；新增 Ctrl+Enter 快捷提示文字；对话框居中显示

## v1.0.0 (2026-02-15)

### ✨ 功能
- PubMed E-utilities + Crossref REST API 双源检索
- 中文关键词自动翻译（环境毒理学/流行病学领域 100+ 术语映射）
- 智能查询构建：中文自然语言描述 → 英文检索式
- DOI 格式校验 + Crossref API 在线验证
- 研究类型过滤：动物实验/人群研究/综述文献筛选
- 年份范围过滤
- 结果去重合并（按 DOI / 标题）
- CSV / TXT 导出（含摘要可选）
- tkinter 图形界面，现代化配色设计
- 命令行模式 (CLI) 支持
