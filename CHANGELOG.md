# LitSearch 更新日志

## v1.2.0 (2025-07-16)  —  精准检索升级

### 🎯 检索质量大幅提升（P0）
- **PubMed 同义词扩展**: 关键词自动匹配 `PUBMED_SYNONYM_GROUPS`，以 OR 组替换单词，避免多词 AND 导致返回 0 结果。例如 "PFAS neurotoxicity pregnancy" → 三个 OR 概念组 AND 连接，每组包含 5-15 个同义词
- **多维度相关性评分 v2**: 替换原有扁平 TF-IDF，改为 6 维度规则评分（总分 100）：
  - 污染物匹配 (30分) — 精准锚定课题核心物质，非目标暴露物得 0 分
  - 健康结局 (25分) — 神经/内分泌/免疫/代谢/癌症 5 类结局词库
  - 暴露窗口 (15分) — 孕期/围产期/哺乳期/儿童期/成年期
  - 研究对象 (10分) — 人群/动物/细胞
  - 机制/方法 (10分) — 氧化应激、Western blot 等
  - TF-IDF 语义 (10分) — 保留原有兜底能力
- **课题防护栏 (Topic Guardrails)**: PFAS 课题自动启用专项规则 —— 命中核心污染物 → 优先展示；命中非目标暴露物（nicotine/morphine 等）→ 大幅降权；仅泛环境综述 → 小幅降权

### 🏷 结构化标签（P1）
- **文献自动打标**: 新增 `paper_tagger.py`，根据标题+摘要自动识别 3 类标签：
  - 研究类型（流行病学/动物实验/体外实验/综述/其他）
  - 污染物类别（PFAS/重金属/其他环境污染物/未知）
  - 暴露窗口（孕期/围产期/哺乳期/儿童期/成年期/未知）
- 标签在表格、详情窗口、CSV/TXT 导出中均可见

### 📦 新增模块
- `domain_vocab.py` — 集中维护 PFAS 同义词、污染物词典、暴露窗口、结局类型、机制术语等（~340 行）
- `relevance_analyzer.py` v2 — 多维度评分引擎（~430 行）
- `topic_guardrails.py` — 课题防护栏（参数化设计，支持扩展到重金属/农药等课题）
- `paper_tagger.py` — 文献结构打标

### 🔧 改进
- `models.py` 新增 `research_type`, `pollutant_category`, `exposure_window` 字段
- `exporter.py` CSV/TXT 导出包含结构标签
- `gui.py` 表格新增 3 列（研究类型/污染物/暴露窗口），详情窗口展示标签
- `streamlit_app.py` 表格和卡片视图均展示结构标签
- `query_builder.py` PubMed 查询支持同义词 OR 扩展，AND 组数限制≤4

## v1.1.0 (2026-03-02)

### 🖥 跨平台支持
- **macOS 应用打包**: 新增 `LitSearch_macOS.spec` 配置，支持生成 `.app` 应用程序包（同时支持 Intel 和 Apple Silicon）
- **GitHub Actions 自动构建**: 推送 tag 后自动构建 macOS `.dmg` + Windows `.exe`，并发布到 GitHub Releases
- macOS 首次打开需右键 → 打开（绕过 Gatekeeper）

### 🌐 网页版部署
- **Streamlit 网页版**: 新增 `streamlit_app.py`，支持通过浏览器访问，无需安装任何软件
- 支持 Streamlit Community Cloud 免费部署，任何平台（Mac/Windows/Linux/手机）皆可使用
- 网页版包含全部功能：双源检索、影响因子、相关性分析、智能摘选、CSV/TXT 导出
- 提供表格视图 + 卡片视图两种浏览模式
- 新增 `.streamlit/config.toml` 主题配置

### ✨ 新功能
- **期刊影响因子 (IF)**: 通过 OpenAlex API 自动获取期刊 2 年平均被引次数（等效 JCR Impact Factor），结果展示影响因子数值与等级标识
- **课题相关性分析**: 用户可输入自己的课题描述，系统自动计算每篇文献与课题的相关度分数 (0-100)，基于 TF-IDF 余弦相似度与关键词覆盖率算法
- **智能内容摘选**: 从摘要中自动提取与用户课题最相关的句子，生成参考建议，指出文献可为用户研究提供哪些方面的帮助（研究方法、统计分析、生物指标、健康效应等）
- **文献详情增强**: 详情窗口新增影响因子、相关度评分、智能分析报告
- **版本信息与联系方式**: 界面底部显示版本号、开发者邮箱联系方式
- **开发者署名**: 顶栏与底栏显示开发者标识 "ArcanaJ"，联系信息字体放大至 10pt

### 🔧 改进
- 数据模型新增 `impact_factor` 和 `relevance_score` 字段
- CSV/TXT 导出支持影响因子和相关度信息
- 结果表格新增影响因子列和相关度列，支持排序
- 统计栏增加影响因子均值和相关度均值显示
- 检索输入框和课题描述框改为弹出式对话框输入，点击预览区域弹出大尺寸编辑窗口，支持滚动条、字数统计、Ctrl+Enter 快捷确认
- **UI 全面自适应**：左右面板改用 PanedWindow 可拖拽分割，用户可自由调整左右比例
- 左侧控制面板内容宽度随面板拖拽自动适配，预览标签文字换行宽度动态调整
- 表格标题/作者/期刊/DOI 列设为可拉伸 (stretch)，随窗口变大自动填充空间
- 最小窗口尺寸降至 800×550，支持更小屏幕使用
- 左侧控制面板改为可滚动布局，窗口较小时也能访问所有选项
- 移除检索输入区的"示例"快捷标签，精简界面
- 数据源选项改为竖排排列，每个选项独占一行，可正常点击"仅 PubMed"和"仅 Crossref"
- 结果数量新增 Spinbox 自定义输入（1-100），快捷按钮保留 10/20/30/50/100
- **弹出对话框布局修复**：按钮区域改为先于文本区 pack（side=BOTTOM），确保"✅ 确认保存"、"🗑 清空"、"取消"按钮始终可见；按钮更名为"确认保存"，增大点击区域；新增 Ctrl+Enter 快捷提示文字；对话框居中显示
